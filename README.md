## 众筹合约
本样例演示众筹合约的基本流程，包括部署合约，合约，启动众筹项目，

## 操作流程
- 配置私钥  
在 .env 中放入的私钥，格式为 "PRIVATE_KEY=xxxx", 然后代码自动从中读取

- 安装依赖
```
yarn
```

- 编译合约
```
npx hardhat compile
```

- 测试合约
```
npx hardhat test
```

- 部署合约
```
npx hardhat run scripts/deploy_crowdfunding.js --network kovan
```
